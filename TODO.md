# ODYSCARE Website Development Plan

## Project Overview
Professional multi-page website for ODYSCARE — coffee machine spare parts & maintenance startup.

## File Structure Plan

### HTML Pages
1. `index.html` — Landing Page (animated entry, distribution map, stats, services)
2. `company-profile.html` — Company Profile & History
3. `sekolah-online.html` — SekolahOnline / Tech Lab
4. `gallery.html` — Coffee Machine Maintenance Gallery
5. `partners.html` — Partner Companies
6. `maintenance.html` — Maintenance Form & Live Chat
7. `social.html` — Social Media Hub (WA, Instagram)

### CSS Files
8. `css/styles.css` — Main stylesheet with responsive design
9. `css/animations.css` — Coffee-themed animations & keyframes

### JavaScript Files
10. `js/main.js` — Core interactivity (nav, forms, gallery, chat)
11. `js/animations.js` — Landing animations, particle effects

### Assets
12. `assets/images/` — Directory for images (logos, gallery, team, partners)

## Features Checklist
- [ ] Unique animated landing entry (coffee distribution visual)
- [ ] Professional responsive design (mobile-first)
- [ ] Gallery with filterable coffee machine maintenance photos
- [ ] SekolahOnline tab/page (tech imagination lab)
- [ ] Company profile & history timeline
- [ ] Social media links (WhatsApp, Instagram)
- [ ] Maintenance request form with validation
- [ ] Live chat widget for consultation
- [ ] Partner company showcase
- [ ] Additional professional elements (loading screen, cookie consent, SEO, smooth scroll)

## Brand Colors (Coffee Theme)
- Primary Dark: #2C1810 (dark coffee)
- Primary Medium: #5D4037 (medium roast)
- Accent Gold: #C9A227 (premium gold)
- Cream/Light: #F5F0E8 (milk foam)
- Background: #1A1A1A (dark mode professional)

## Next Steps
Create all files following the plan above, starting with shared assets (CSS/JS) then HTML pages.
