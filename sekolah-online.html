Lab Imajinasi Teknologi

SekolahOnline ODYSCARE

Program akselerasi kompetensi teknologi kopi. Pelajari IoT, sensor suhu, otomasi katup tekanan, cloud integration, dan pemrograman Arduino dari tingkat pemula hingga mahir.

Kurikulum Pilihan

Program Pembelajaran Kami

Program kelas dirancang khusus untuk memadukan pemahaman kopi dengan rekayasa teknologi perangkat keras.

Arduino for Coffee Tech

Pelajari dasar elektronika, pembacaan sensor suhu (RTD/Thermocouple), kontrol pemanas (SSR), dan antarmuka LCD untuk mesin espresso.

IoT Coffee Brewing

Mengintegrasikan ESP32 dengan aplikasi Blynk atau Web Dashboard untuk memantau data penyeduhan kopi (suhu, debit air, TDS) dari jarak jauh.

Otomasi & PLC Roaster

Pemrograman automasi profil roasting menggunakan PLC sederhana atau Arduino Mega untuk mengontrol kecepatan putaran motor drum dan damper udara.

Cloud & Data Analytics

Membaca tren data operasional beberapa mesin sekaligus di cloud server (AWS/Google Cloud) untuk mendeteksi degradasi performa mesin kopi.

Gabung Sekarang

Pendaftaran Kelas Teknologi Kopi

Mulai langkah Anda sebagai Teknisi Kopi Modern masa kini. Isi formulir pendaftaran untuk mengajukan beasiswa atau pendaftaran mandiri. Kami menyediakan Hardware Kit lengkap yang akan dikirim langsung ke rumah Anda!

Termasuk Starter Kit Hardware

Setiap siswa mendapat modul ESP32, Sensor Suhu DS18B20 presisi, Layar OLED, relay, kabel jumper, dan modul pendukung.

Sertifikat Resmi Kelulusan

Dapatkan sertifikasi profesional teknisi IoT industri kopi yang diakui oleh mitra distributor mesin kopi kami setelah lulus ujian proyek akhir.
